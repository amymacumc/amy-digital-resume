# Amy Digital Resume

Personal resume and portfolio website for Amy, positioned as a Digital Project Manager, product-minded Business Analyst, and cross-functional digital delivery owner.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

For GitHub Pages static export:

```bash
GITHUB_PAGES=true npm run build
```

The editable profile content lives in `src/data/data.tsx`.
